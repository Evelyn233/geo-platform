# 真实AI大模型数据采集平台 - 项目总结

## 🎯 项目概述

我已经为您创建了一个完整的**真实AI大模型数据采集平台**，可以采集各种AI大模型对您网站的真实曝光数据，并提供类似图片中仪表板的专业分析功能。


## 🚀 核心功能

### 1. 真实数据采集
- ✅ **多平台支持**: 豆包、Deepseek、通义、元宝、KIMI等
- ✅ **API集成**: 直接调用各AI平台官方API
- ✅ **智能问题生成**: 基于关键词自动生成相关问题
- ✅ **速率限制**: 智能控制API调用频率
- ✅ **错误重试**: 自动重试失败请求

### 2. 数据分析与可视化
- ✅ **曝光率统计**: 各平台问题分布和曝光率
- ✅ **关键词分析**: 热门关键词频率统计
- ✅ **设备分布**: 电脑端vs移动端对比
- ✅ **专业仪表板**: 类似图片中的统计界面
- ✅ **实时图表**: 饼图、柱状图等可视化

### 3. 任务调度系统
- ✅ **定时采集**: 支持cron表达式定时任务
- ✅ **任务队列**: 多任务并发处理
- ✅ **状态监控**: 实时任务状态跟踪
- ✅ **错误处理**: 自动重试和错误恢复

### 4. 用户界面
- ✅ **专业仪表板**: 类似图片中的分析界面
- ✅ **真实数据采集**: 输入关键词和选择平台
- ✅ **模拟数据生成**: 快速生成测试数据
- ✅ **数据搜索**: 支持关键词和平台筛选
- ✅ **数据导出**: JSON/CSV格式导出

## 🎮 使用方法

### 1. 快速开始
```bash
# 1. 配置API密钥
编辑 ai_platform_config.json，填入您的API密钥

# 2. 启动系统
python geo_app.py

# 3. 访问界面
打开 http://localhost:8000

# 4. 开始采集
点击 "AI曝光率分析" → 输入关键词 → 选择平台 → 开始采集
```

### 2. 配置API密钥
```json
{
  "platforms": [
    {
      "name": "豆包",
      "api_key": "your_actual_api_key_here",
      "enabled": true
    }
  ]
}
```

### 3. 数据采集流程
1. **输入关键词**: 在文本框中输入要分析的关键词
2. **选择平台**: 勾选要采集的AI平台
3. **开始采集**: 点击"开始真实采集"按钮
4. **查看结果**: 在仪表板中查看采集的数据和分析

## 🔌 API接口

### 数据采集接口
- `POST /api/real_data/collect` - 采集真实数据
- `GET /api/real_data/platforms/status` - 检查平台状态
- `POST /api/real_data/analyze` - 分析数据
- `POST /api/real_data/export` - 导出数据

### 任务调度接口
- `POST /api/real_data/scheduler/create_task` - 创建采集任务
- `GET /api/real_data/scheduler/status` - 获取调度器状态

## 📋 配置说明

### AI平台配置 (ai_platform_config.json)
```json
{
  "platforms": [
    {
      "name": "平台名称",
      "api_base_url": "API基础URL",
      "api_key": "API密钥",
      "rate_limit": 20,
      "timeout": 30,
      "enabled": true
    }
  ]
}
```

### 调度器配置 (scheduler_config.json)
```json
{
  "worker_count": 3,
  "max_queue_size": 100,
  "task_timeout": 300,
  "scheduled_tasks": [
    {
      "name": "每日采集",
      "keywords": ["关键词1", "关键词2"],
      "platforms": ["平台1", "平台2"],
      "cron": "0 9 * * *",
      "enabled": true
    }
  ]
}
```
